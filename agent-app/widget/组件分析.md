# Widget 组件现状

本文档用于快速对齐 `sdk/agent-app/widget` 当前实际支持的组件能力，避免继续参考过期结论。

## 当前已支持的组件

| 组件类型 | 文件 | 常用配置 | 说明 |
|----------|------|----------|------|
| `ID` | `id.go` | — | 只读主键/标识；类型值区分大小写，必须写 `ID` |
| `input` | `input.go` | `placeholder`、`prepend`、`append`、`render_default` | 单行文本输入；不支持 `password:true` |
| `text` | `text.go` | `format` | 只读文本展示，支持 `json/yaml/xml/markdown/html/csv` 等 |
| `text_area` | `text_area.go` | `placeholder`、`render_default` | 多行文本输入 |
| `richtext` | `richtext.go` | `height` | 富文本编辑 |
| `select` | `select.go` | `options`、`options_colors`、`placeholder`、`render_default`、`creatable` | 下拉单选 |
| `radio` | `radio.go` | `options`、`render_default` | 单选按钮 |
| `checkbox` | `checkbox.go` | `options`、`render_default` | 固定选项复选 |
| `multiselect` | `multiselect.go` | `options`、`options_colors`、`placeholder`、`render_default`、`max_count`、`creatable` | 下拉多选 |
| `list` | `list.go` | `item_type`、`separator`、`placeholder`、`render_default`、`unique`、`max_count` | 自由输入列表；`item_type:number` 适配 `[]int` / `[]float`，`item_type:text` 适配 `[]string` |
| `integer` | `integer.go` | `placeholder`、`min`、`max`、`step`、`render_default`、`unit` | 整数输入 |
| `float` | `float.go` | `placeholder`、`min`、`max`、`precision`、`step`、`render_default`、`unit` | 小数输入 |
| `slider` | `slider.go` | `min`、`max`、`step`、`render_default`、`unit` | 可编辑滑块 |
| `rate` | `rate.go` | `max`、`allow_half`、`render_default`、`texts` | 星级评分 |
| `switch` | `switch.go` | `render_default` | 布尔开关 |
| `datetime` | `datetime.go` | `format`、`render_default`、`disabled` | 日期时间；raw value 为 `"YYYY-MM-DD HH:mm:ss"`，数据库推荐真实时间类型 |
| `color` | `color.go` | `format`、`render_default`、`show_alpha` | 颜色选择 |
| `files` | `files.go` | `accept`、`max_size`、`max_count` | 文件上传/下载 |
| `user / users` | `user.go` / `users.go` | `render_default`、`disabled`、`max_count` | 用户选择 |
| `department / departments` | `department.go` / `departments.go` | `render_default`、`max_count` | 部门选择 |
| `progress` | `progress.go` | `min`、`max`、`unit` | 只读进度展示 |
| `link` | `link.go` | `text`、`target`、`link_type`、`icon` | 只读跳转链接；`type` 是组件类型保留 key，链接样式使用 `link_type` |
| `table` / `form` | `decode.go` / `widget.go` | — | 嵌套结构体容器 |

## 关键约定

- `type` 大小写敏感，主键展示写 `type:ID`，不要写成 `type:id`。
- 当前没有独立 `date`、`time`、`range`、`image`、`tag`、`tree`、`cascader`、`code` 等组件；完整日期时间统一使用 `datetime + types.Time`。
- `render_default` 是前端渲染默认值：只在新增/初始化界面时填充，不等于 `gorm:"default:..."` 或数据库默认值。
- `switch` 当前只支持 `render_default`，不支持 `true_label`、`false_label` 这类配置。
- `text`、`progress`、`link`、`ID` 更适合只读字段或响应字段。
- `table` / `form` 只支持具名 `struct` 或 `[]struct` 递归解析，不支持 `map[string]interface{}`。
- 数据类型通常由 Go 字段类型推断；`files` 和 `datetime` 会按前端/API 协议输出为 `string`。
- 字段只要写了 `widget` 标签，就必须显式配置 `type`；同级字段的 `json` code 不能重复。
- SDK 启动期会校验 widget tag 白名单；拼错的配置键会直接失败，例如 `optoins`、`maxcount`、`linkType`。
- `input` 不支持 `password:true`。密码、token、API key 等如需存储，按普通业务字段处理；需要避免进入平台操作日志时写 `sensitive:"true"`。`sensitive` 不表示业务库加密。
- Table 筛选字段写在 Request 中，并由 Handler 显式处理查询条件。
- `data` tag 只允许 `format`、`example`；`display` tag 只允许 `scenes`，且 scenes 只能是 `list/create/update`，不能为空、不能重复。
- `table` / `form` 容器不能配置 `hide:"create,update"`；容器只能出现在表单/嵌套结构中，不作为列表列展示。
- `integer`、`float`、`slider`、`rate` 的 `render_default` 必须落在组件 `min/max` 范围内；`slider` 未配置范围时按默认 `0-100`，`rate` 未配置 `max` 时按默认 `5`。
- `list` 的 `render_default` 会按 `item_type` 校验：`item_type:number` 的默认值每项必须是数字，默认值数量不能超过 `max_count`。
- 纯静态 `multiselect` 在没有 `creatable:true`、没有 `OnSelectFuzzyMap` 时，`max_count` 不能超过 `options` 数量。
- `color` 的 `render_default` 会拦截明显非法值，支持 hex、`rgb(...)` / `rgba(...)` 和少量常见颜色名。
- `datetime` 的 `render_default` 支持静态时间字符串，或 `CURRENT_TIMESTAMP`、`CURRENT_DATE`、`DATE_ADD/DATE_SUB(..., INTERVAL n UNIT)` 这类前端可解析表达式。
- 字段级 `callback` 输出会 trim 并去重；`OnSelectFuzzy` 仍必须配套 `OnSelectFuzzyMap`。
- `DecodeTable` 支持 `PageSortReq` 链路下 Request 显式声明与 Model 同名的筛选字段。

## 系统审计字段约定

这些字段属于项目里约定俗成的系统字段，SDK 会按字段 `json` 名称做启动期校验，避免前端 schema 和 GORM 落库语义漂移。

- `id`：仅当 `gorm` 带 `primaryKey` / `primary_key` 时按主键约束；必须写 `widget:"name:ID;type:ID"`、`hide:"create,update"`，并且 `gorm` 必须包含 `autoIncrement` 和 `column:id`。
- `created_at`：必须是 `datetime` 组件，格式固定为 `YYYY-MM-DD HH:mm:ss`，展示固定为 `hide:"create,update"`；`gorm` 必须包含 `column:created_at` 和 `autoCreateTime`。
- `updated_at`：必须是 `datetime` 组件，格式固定为 `YYYY-MM-DD HH:mm:ss`，展示固定为 `hide:"create,update"`；`gorm` 必须包含 `column:updated_at` 和 `autoUpdateTime`。
- `created_by` / `updated_by`：必须使用 `user` 组件，展示固定为 `hide:"create,update"`；`gorm` 的 `column` 必须和字段 `json` 名称一致。
- `deleted_at` / `deleted_by`：软删除字段不应该进入前端 schema，必须使用 `widget:"-"` 或 `json:"-"` 隐藏。
